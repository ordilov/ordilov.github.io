## 정적 페이지 배포하기
동적으로 바뀌는 사이트가 아니라면 github pages를 통해 호스팅이 가능합니다.  

## 설정하기
Settings > Pages > Source  
배포하길 원하는 브랜치를 설정해줍니다.  

## 배포 확인하기
Environments > github-pages  
현재 배포중인 상태를 확인 가능합니다.  
View deployment를 통해 웹사이트로 이동할 수 있습니다.
