Jeykll 목차 추가하기

ordilov / 1/6/202

TOC (Table of Contents)

목차를 영어로 TOC라고 부릅니다.

gem 설치로 추가하기
plugin 추가하기

Gemfile에 아래 명령어를 추가해줍니다.

gem 'jekyll-toc'
_config.yml 추가하기

플러그인 목록에 jekyll-toc를 추가해줍니다.

plugins:
  - jekyll-toc
toc 설정 추가하기

목차를 추가할 post 설정에 toc: true를 추가합니다.

---
layout: post
toc: true
---
사용하기

사용할 layout html에 toc를 추가해야합니다.

전체에 다 적용될려면 default.html 에{{ content | toc }} 를 추가합니다.

{{ content | toc }}

content와 연결된 형태가 아닌 따로 사용하는 경우 변수로 사용합니다.

gem 설치 없이하기
toc html 받아오기

jekyll toc 에서 toc.html을 다운받습니다.
toc.html파일을 _includes 폴더에 넣어줍니다.

사용하기

사용할 html에 다음 태그를 추가해줍니다.

{% include toc.html html=content %}
toc를 옆에 추가하기

toc를 옆에 추가하려면 <aside>에 넣어주면 됩니다.

<aside>
    {% include toc.html html=content %}   
</aside>
